# xiaxiezhou
这是一个从个人兴趣慢慢积累出来的项目，项目中文名是“虾蟹粥”，它是一个AI工程，它聚焦在如何搭建企业级的上下文工程，充分激发大模型完成任务的潜力。This is a project that has gradually grown out of personal interest. The project's Chinese name is "虾蟹粥" (Xiā Xiè Zhōu). It is an AI engineering effort focused on how to build enterprise-level context engineering to fully unleash the potential of large language models in completing tasks.
